# 预组方向

## 上头组 / 爆杀流

- **关键词：** 强攻、连杀、爆发、卖血、退场。
- **玩法：** 找【杀】、强化【杀】、造成伤害、触发 Mega、继续压血。
- **风险：** 怕断【杀】，怕被【闪】，资源消耗大。

## 密裁组 / 密裁

- **关键词：** 情报、控制、弃牌、节奏。
- **玩法：** 观看牌堆或手牌→触发本体→精准弃牌→把情报转化为进攻窗口。
- **风险：** 直接伤害有限，只看不打会让对手获得喘息机会。

## 操作组 / 锦囊流

- **关键词：** 锦囊、资源、支援、牌堆操作、回收、连锁。
- **玩法：** 找锦囊→使用锦囊→触发本体摸牌→控牌堆→回收→继续操作。
- **风险：** 怕快速压血、怕断锦囊、依赖角色区、伤害不稳定。

## 变通组 / 拟态流

- **关键词：** 转化、拟态、宣言、灵活、变通。
- **玩法：** 宣言转化→视为打出→临时暗置上阵角色→继续发动技能→Mega后降低临时角色的【休整X】费用。
- **风险：** 操作链长、怕被弃牌干扰、对手牌数量要求高。

## 执棋组 / 调度流

- **关键词：** 识局、布阵、明置、换阵、角色牌堆。
- **玩法：** 明置角色→整理角色牌堆→主动补位→Z招式就绪→一次性重构己方阵容。
- **风险：** 依赖角色区和角色牌堆，本体与Z招式均不直接提供稳定伤害。

## 逆命组 / 卖血流

- **关键词：** 卖血、判定、资源、反击、续航。
- **玩法：** 承受伤害→本体判定→利用红黑结果→控制并回复血线→Z招式回收弃牌区关键牌。
- **风险：** 血线危险，惧怕高额爆发；直接失去体力不能触发受到伤害后的技能。

## 幽幕组 / 伏击流

- **关键词：** 伏击、暗置、响应、补位、退场区。
- **玩法：** 暗置伏击→响应对手行动→支付费用→本体暗置补位→Z招式调用退场区伏击。
- **风险：** 依赖对手触发时机，公开的伏击可能被绕开，非伏击角色过多会降低补位质量。

## 不落组 / 防御流

- **关键词：** 防御、闪、减伤、保护、续航。
- **玩法：** 抵消【杀】→防止或减少伤害→本体摸牌→保护己方角色→Z招式抵御致命伤害。
- **风险：** 主动压血能力有限，直接失去体力可以绕过伤害防御，过度防守会错失反击窗口。

## 运营组 / 制衡流

- **关键词：** 资源、换牌、休整循环、防御、稳定。
- **暂未制作。**
