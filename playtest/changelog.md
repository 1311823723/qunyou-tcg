# Changelog

## 2026-06-20 角色标签规范化

- 将角色牌标签从104种近义或一次性标签收敛为27种正式机制标签。
- 每张角色牌最多保留2个标签；移除爆发、压制、续航、应变等强度或风格描述，改用技能实际涉及的机制。
- 新增 `data/tag-taxonomy.json`，并在卡牌校验中检查标签白名单、数量上限和单卡重复标签。

## 2026-06-20 第六至第八预组与120张角色目标

- 新增逆命组 / 卖血流预组，本体为【风妖精】，围绕受到伤害、红黑判定、弃牌利用和低血回收展开。
- 新增幽幕组 / 伏击流预组及本体【小阿潘】；本体技能【暗潮接替】在伏击支付休整或退场费用后暗置补位，Z招式【万劫暗夜·亡者伏阵终焉再临】可限次调用退场区伏击。
- 新增不落组 / 防御流预组及本体【天天】；本体技能【守势循环】将成功防御转化为摸牌，Z招式【绝境神铠·不灭灵魂究极再临】可防止一次致命伤害并回复2点体力。
- 新增34张角色牌，编号为 `char_087` 至 `char_120`；新增牌名称均采用《鹅鸭杀》身份与指定群友名组合。
- 角色牌总数由86张增至120张，强攻、防御、资源、控制、支援、伏击六种主定位各20张。
- 新增卖血流判定联动、伏击补位与退场区调用、防御成功收益和复合费用反制等技能设计。

- 新增6张角色牌：【变形鸭-爱吃豚侠】【入殓师-爱吃豚侠】【侦探-爱吃豚侠】【观者-柯柯】【保镖-爱吃豚侠】【警长-爱吃豚侠】。
