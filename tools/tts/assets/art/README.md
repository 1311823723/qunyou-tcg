# TTS art assets

These images are used only by the local Tabletop Simulator export pipeline.

Current mappings are defined in `tools/tts/export-tts.js`:

- `keke-assassin.png`: normal Keke assassin art for `body_mizai_001` and `char_aggro_001`.
- `keke-assassin-mega.png`: Mega-style Keke art for the back face of `body_mizai_001`.

Card data remains in `data/cards/*.json`; art mapping is export-only.
